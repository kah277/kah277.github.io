For my capstone course in spring 2022, I collaborated with a team of students to create a web-based dashboard application for Reddit users who participate in addiction recovery support subreddits. My group and I utilized several different tools and technologies for this project, many of which were brand new to us. We built the dashboard using Python, the Reddit API and PRAW, HTML, CSS, Flask, JavaScript, and PostgreSQL.
