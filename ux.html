UX Design Process

Below is a group project from my User-Centered Design course. We were tasked with designing a mobile app for a fictional arcade and restaurant.

We started by writing personas and user scenarios in order to gather user requirements, and then created wireframes (using Balsamiq) based on those requirements.

We filled out a style guide template provided by our instructor, and then started building a prototype using Figma. After our initial design, we conducted user testing with other students in the class. We made final changes to our design based on input from user testing.

In December 2023, I revisited this design and I am now in the process of redesigning it. I've added some context to the home screen by adding some intro text, and I've also redesigned the category sections to make them look more button-like. I've changed the background to a light cream color to create more negative space, and decided to incorporate the green accent color in the "register" button instead.
